# 🚀 LaTF - 官方项目标识

**项目名称：** LaTF

**完整标题：** LaTF：图像真实性检测的混合深度学习系统

**子标题：** AI生成图像与篡改检测的一体化解决方案

---

## 📌 为什么叫 LaTF？

该项目融合了两种核心深度学习技术：
- **La**RE：来自扩散模型的潜在重建误差 (Latent Reconstruction Error)
- **T**ru**F**or：基于局部痕迹的篡改检测 (Detecting Image Forgeries by Localized Traces)

结合起来形成**混合双层检测系统**：
1. **全局证据** - CLIP语义特征
2. **局部证据** - LaRE重建误差 + TruFor可靠性图

---

## 🎯 核心特性

- ✓ **双层检测**：全局分类 + 局部篡改定位
- ✓ **科学融合**：基于证据的混合方法
- ✓ **生产就绪**：V13模型在多个AI生成器和篡改数据集上训练完毕
- ✓ **智能采样**：重采样珍贵的doubao数据，欠采样大量的flux/SDXL数据
- ✓ **级联推理**：自适应两阶段推理，提高效率
- ✓ **多检测器融合**：CLIP + LaRE + TruFor + 统计集成

## 🚀 快速开始

```bash
# 1. 环境配置
conda create -n latf python=3.9
conda activate latf
pip install -r requirements.txt

# 2. 生成注释
python script/1_gen_annotations.py --output_name sdxl

# 3. 提取LaRE特征
python script/2_extract_features.py --input_path annotation/train_sdxl.txt

# 4. 生成TruFor图
python script/5_gen_trufor_maps.py

# 5. 训练模型（V13）
python script/5_train_model_v11.py --enable_trufor true

# 6. 启动Web服务
python web/flask/app.py
```
