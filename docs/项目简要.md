# 🚀 LaTF - 官方项目标识

**项目名称：** LaTF

**完整标题：** LaTF：图像真实性检测的混合深度学习系统

**子标题：** AI生成图像与篡改检测的一体化解决方案

---

## 📌 为什么叫 LaTF？

LaTF 沿用项目历史缩写，当前指代以 LaRE 为核心、融合取证定位能力的图像真实性检测系统。

当前主线形成**混合双层检测系统**：
1. **全局证据** - CLIP语义特征
2. **局部证据** - LaRE重建误差 + 亮度/掩码引导定位

---

## 🎯 核心特性

- ✓ **双层检测**：全局分类 + 局部篡改定位
- ✓ **科学融合**：基于证据的混合方法
- ✓ **当前主线**：V14多尺度定位训练链路已替代旧 TruFor 预生成流程
- ✓ **智能采样**：重采样珍贵的doubao数据，欠采样大量的flux/SDXL数据
- ✓ **级联推理**：自适应两阶段推理，提高效率
- ✓ **多检测器融合**：CLIP + LaRE + SSFR/FLH + 统计集成

## 🏆 生产模型

**当前主线版本：** V14（v14_multiscale）
- **架构：** 多分支融合（CLIP + ConvNeXt + LaRE + 多尺度定位头）
- **目标：** 在保住分类性能的前提下增加 32/64/128 多尺度定位能力
- **训练特征：** 采用 freeze_loc_epochs、亮度引导和掩码监督
- **状态：** 当前训练主线

## 🚀 快速开始

```bash
# 1. 环境配置
conda create -n latf python=3.9
conda activate latf
pip install -r requirements.txt

# 2. 生成注释
python script/1_gen_annotations.py --output_name sdxl

# 3. 提取LaRE特征
python script/2_extract_features.py --input_path annotation/train_sdxl.txt

# 4. 训练模型（V14）
python script/5_train_model_v11.py --use_amp

# 5. 按类别评估
python test/evaluate_by_category.py --model outputs/v14_multiscale/best.pth

# 6. 启动Web服务
python web/flask/app.py
```
